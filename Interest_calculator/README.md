# 💰 Simple Interest Calculator

A beginner-friendly Python program that calculates **simple interest** and the **total amount after a given time**.
This project demonstrates basic Python concepts such as **loops, input validation, and arithmetic operations**.

---
## 📌 Description

The program asks the user to enter:

* Initial amount (principal)
* Interest rate (%)
* Time in years

It ensures that all values entered are **greater than zero** using `while` loops.
After receiving valid input, it calculates:

* **Total amount after interest**
* **Interest earned**
---

## 🧮 Formula Used

Simple Interest Formula:

```
Total Amount = Principal × (1 + (Rate × Time) / 100)
```
```
Interest Earned = Total Amount − Principal
```
---
## 🚀 Features

* User input validation
* Prevents zero or negative values
* Uses loops to ensure correct input
* Displays results with rounded values
* Beginner-friendly Python code

---

## ▶️ How to Run the Program

1. Install **Python 3**
2. Download or clone the repository
3. Run the program in the terminal

```
python interest.py
```
